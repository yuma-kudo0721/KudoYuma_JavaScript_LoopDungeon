# ループダンジョン

## 概要
ランダム生成ダンジョンを探索するローグライク風プロトタイプ。  
JavaScriptとCanvasを用い、ターン制処理・視界制限・簡易敵AIなどの
**ゲームシステム設計および拡張性の検証**を目的として制作。

## 実装済み要素
- ランダムダンジョン生成
- 視界制限（View Radius）
- 敵AI（簡易追跡）
- アイテム管理（回復・能力強化）
- 固定マップのショップエリア

## 工夫した点
- Dungeon / ShopFloor を分離し、マップ切り替えを容易にする設計
- Item クラスに効果データを持たせ、処理を共通化
- 視界外タイルの描画を制御し、探索性を意識した表現を実装
